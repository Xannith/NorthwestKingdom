Photo metadata and permissions

All photos published here include the following information where available:

  • Title — a descriptive name
  • Date — when the photo was taken
  • Location — general location in NWK
  • Photographer — name or "Anonymous" if preferred
  • Permission status — confirmed public use
  • Caption — brief description
  • Tags — topic, year, event, etc.
  • Related event — linked to calendar or archive entry, if applicable

Photos awaiting permission review, or approved only for member viewing, are held in the member-only area and are not publicly accessible.
