PDF Conversion Queue

Track PDFs queued for conversion to searchable HTML. Manage conversion status, metadata, and review state.

Scaffold only. This tool will track uploaded PDFs and their conversion status. See DOCUMENT_CONVERSION_PLAN.md for the conversion workflow and naming conventions.

Conversion workflow

  1. Upload PDF to the appropriate location (public or member area)
  2. Add entry here with status: Needs Conversion
  3. Convert PDF to HTML (OCR if scanned)
  4. Review HTML output for accuracy
  5. Set status to Needs Review
  6. After review, mark as HTML Available and update the document page
  7. Add to search index (public or member, not both)
